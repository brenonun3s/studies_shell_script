
# README - Shell Script Studies

Este repositório contém exemplos e exercícios de **Shell Script** para fins de aprendizado e prática.

## 📚 Conteúdo

Coleção de scripts em bash/shell para estudar conceitos como:
- Variáveis e tipos de dados
- Estruturas de controle (if, for, while)
- Funções
- Manipulação de strings e arquivos
- Processamento de argumentos
- Pipes e redirecionamento

## 🚀 Como usar

```bash
# Clonar o repositório
git clone https://github.com/brenonun3s/studies_shell_script.git

# Navegar até o diretório
cd studies_shell_script

# Executar um script
bash nome_do_script.sh
```

## 📝 Estrutura

```
scripts/
├── basicos/
├── intermediarios/
└── avancados/
```
